## React 설치하기

```
$npx create-react-app {프로젝트명}
```

위 명령어 한 줄로 아주 간단하게 `react` 프로젝트가 생성되었다.


`npm start`명령어를 통해 react 프로젝트가 생성된것을 확인할 수 있다.
